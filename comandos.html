Comandos no terminal

Comandos básicos

ls → lista os arquivos

echo → apresenta mensagem na tela

> → coloca um argumento dentro de um arquivo (ex.: echo “Bem vindo” > bemvindo.txt)

clear → limpa o terminal

ls -la → mostra se é um diretório ou arquivo, usuário, data de criação, tamanho, e arquivos e diretórios ocultos

pwd → mostra o caminho do diretório que estamos

Comandos básicos

cd nomeDiretorio → entra no diretório

cd .. → volta ao diretório anterior

cd → volta para o diretório base

mkdir nomeDir → cria novo diretório

rmdir nomeDir → remove diretórios que estão vazios

cp arq1.txt arq2.txt → copia o conteúdo de um arquivo para outro

man → mostra a documentação referente ao comando (ex.: man ls)

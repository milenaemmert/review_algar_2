Linguagens de programação

Linguagem de alto nível

=> mais próximas da linguagem humana

=> facilidade de aprendizagem

=> maior produtividade

=> manutenção mais fácil

Exemplos: JavaScript, Swift, Python, Java.

Linguagem de baixo nível

=> mais próximas da máquina

=> tempo de processamento mais rápido

=> sintaxe complexa

=> exige maior conhecimento de hardware

Aqui podemos ver a complexidade de um simples "Hello World" sendo escrito em Assembly.

Linguagens interpretadas

=> interpreta e executa a cada linha de código

=> executa o programa em diferentes máquinas

Exemplos: JavaScript, Python, PHP.

Linguagens compiladas

=> verifica erros antes de executar o programa

=> ideal para programas que necessitam de alta perfomance

Exemplos: Go, C, Rust.
